# Uniapp

**1. 页面目录简介：**

1. pages:

   存放我们的页面

2. static:

   存放静态资源 .jpg\\.css\\.js

3. unpackage:

   存放打包后应用文件apk\\ios\\小程序

4. App.vue

   页面的入口文件，完成页面的切换。

5. main.js

   项目的入口文件，项目加载会首先加载main.js文件

6. manifest.json

   App的项目图标啥的

7. pages.json

   设置页面的存放路径和外观管理

8. uni.scss:

   内置的常用样式管理
